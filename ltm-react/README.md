# ltm-react

A React component library for LTM projects.

## Structure

- `src/` - Component library source code
- `frontend/` - Demo frontend that imports and uses the library

## Development

To test the library:

```bash
cd frontend
npm install
npm run dev
```

## Usage

Import components from the library:

```javascript
import { HelloWorld } from 'ltm-react';
```
